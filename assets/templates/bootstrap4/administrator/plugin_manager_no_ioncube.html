In the huge majority of cases your web hosting service will already have the loaders installed, all you need to do is enable them through your Cpanel PHP settings page.

If they haven't, or if you run your own web server then you will need to download the loaders (https://www.ioncube.com/loaders.php) yourself and configure your web server's php.ini file to use them.
