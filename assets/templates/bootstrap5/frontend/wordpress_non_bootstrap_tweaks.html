A bit of a cludge, but it works It's now possible to run Bootstrap 3 within Wordpress default templates but it does require a few little tweaks here and there To ensure that these tweaks don't bork sites that use themes that support Bootstrap (e.g. Leohtian), it's best to put those small tweaks here. If you want a customised version of this file, see http://www.jomres.net/manual/developers-guide-2/68-customising-jomres/customising-templates-layout/306-template-overrides A customised version of this file is recommended if you intend modify it, as versions of this file in the override directories are not overwritten when Jomres is updated.
